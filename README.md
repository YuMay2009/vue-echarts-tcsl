# vue-echarts-tcsl

> 基于echarts4.X的图表组件库 vue-echarts-tcsl


## Install
``` bash
npm install -g vue-echarts-tcsl
```

## Quick Start
``` bash
import Vue from 'vue'
import vEchartsTcsl from 'vue-echarts-tcsl';
Vue.use(vEchartsTcsl);

# In the template
<vue-echarts-tcsl></vue-echarts-tcsl>
```
